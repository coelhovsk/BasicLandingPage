Sobre Data Science

Data science é aquela mistura de estatística, programação e conhecimento do negócio pra extrair insights dos dados. Serve pra negócios, saúde, ciência e muito mais.

Nosso rolê é deixar data science fácil pra quem tá começando.

Projetos Exemplo

Projetos simples pra você começar:

Análise de Dados Básica

Que tal analisar dados do ENEM ou do IBGE? É um ótimo jeito de começar. Usa o Pandas pra limpar os dados e Matplotlib pra criar uns gráficos maneiros.

Previsão de Preços

Pega uns dados de imóveis e tenta prever os preços com regressão linear. Dá pra fazer com Scikit-learn e é bem satisfatório ver o modelo funcionando!

Classificação de Texto

Que tal criar um modelo que classifica se um tweet é positivo ou negativo? Dá pra fazer com NLP básico e uns algoritmos simples.
